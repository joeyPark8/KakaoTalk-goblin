1. 카카오톡 대화 파일 얻기

카카오톡 도깨비는 휴대폰 카카오톡 대화 기록 파일을 사용합니다.

따라서 먼저 휴대폰 카카오톡에서 대화 기록을 가져 오겠습니다.

1.1 휴대폰 카톡에 들어간다.

1.2 분석하고 싶은 단체대화방에 들어간다.

1.3 더보기 메뉴에서 설정에 들어간다.

1.4 대화 내용 내보내기 버튼을 클릭한다.

1.5 모든 메시지 내부저장소에 저장 버튼을 클릭한다.

1.6 카톡을 나와서 내 파일 앱에 들어간다.

1.7 내장 메모리에 들어간다.

1.8 KakaoTalk 폴더에 들어간다.

1.9 Chats 폴더에 들어간다.

1.10 가장 밑에 있는 폴더에 가장 최근에 다운로드한 파일이 있으므로 가장 밑에 있는 폴더를 컴퓨터로 옮긴다.



2. 카카오톡 도깨비 앱 폴더에 파일 넣기

폴더를 컴퓨터로 옮겼다면 이제 그 폴더에 있는 대화파일(txt 파일)을 카카오톡 도깨비 앱 폴더에 넣어야 합니다.

2.1 카카오톡 도깨비 앱이 있는 폴더에 들어간다.

2.2 files 폴더에 들어간다.

2.3 카카오톡 대화파일(txt 파일)을 넣는다.



3. 카카오톡 도깨비 앱과 파일 연결하기

이제 카카오톡 도깨비 앱을 사용할 차례 입니다.

3.1 카카오톡 도깨비 앱을 실행 한다.

3.2 앱이 실행 되었다면 a를 입력하고 엔터를 쳐서 파일 연결 페이지로 들어간다

3.3 파일 이름을 입력해달라고 하면 확장자와 같이 파일 이름을 입력하고 다시 엔터를 친다(예: 만약 파일 이름이 kakaoChats라면 kakaoChats.txt를 입력).

3.4 파일이 정상적으로 연결되었다고 뜨고 메인 페이지로 나와졌다면 파일 연결이 끝났다.

3.5 카카오톡 도깨비는 파일을 기억하는 기능이 있기 때문에 계속 같은 파일을 사용한다면 다음부터는 파일 이름을 적어주지 않아도 된다
